    Placa-mãe

    Definição

    A placa-mãe (ou motherboard, em inglês) é uma placa de circuito impresso que permite interligar todos os componentes de um computador.

    Processador

    Definição

    O Processador é um circuito integrado capaz de efetuar o processamento de dados, obedecendo a um conjunto predeterminado de instruções. Ele se comunica com o seu exterior (memória, chips da placa-mãe, periféricos, etc) através dos barramentos.

    Memória RAM

    Definição

    A memória de acesso aleatório (do inglês Random Access Memory) é um tipo de memória que permite a leitura e a escrita de dados de forma rápida.

    Características:

    • Memória principal: pode ser lida e gravada pela CPU a qualquer instante. A CPU usa para armazenar e executar programas vindos do disco, para ler e escrever os dados que estão sendo processados.
    • Mémoria volátil: seus dados são perdidos após cessar a alimentação; pois todos os dados que não forem guardados de forma permanente serão apagados após encerramento do computador.

    Disco Rígido

    Definição

    O disco rígido (também conhecido como HD, que vem do inglês Hard Disk) não acessado diretamento pelo CPU, diferente da memória RAM. O seu acesso é feito através de interfaces ou controladores especiais.

    Características:

    • Memória secundária: não é necessária para o funcionamento do computador, mesmo sendo muito importante, já que guarda os dados permanentemente.
    • Mémoria não volátil: seus dados permanecem mesmo após cessada a alimentação, isto é, não será apagado, alterado ou perdido quando se desliga o computador.
